  1. Chatbot - Line機器人"企鵝Bar"

    2. 狀態 : 階段完成

    語言 : Python

    學習點 : LineBot架設、Google sheet API、Render託管

    開發目的:

    現代人無法脫離Line功能，既然無法脫離同時還有記帳需求，可以使用LineBot機器人使得使用者可以用傳訊息方式進行記帳，若需要觀看整體記帳內容也可以進入GoogleSheets進行觀看，同時GoogleSheets為資料儲存在個人之Google帳號，能夠有效的保障資料的儲存。

    • 開發環境 : Visual Studio Code
    • 開發語言 : Python
    • 原碼位置 : GitHub
    • 資料庫 : Google sheet
    • 網址轉址 : Render
    • 訊息API : Line

  3. App - "企鵝記帳去"

    4. 狀態 : 規劃中

    語言 : Flutter

    預計學習點 : Flutter、Google Firebase、Git版本控制

    開發目的:

    本應用程式的開發目的在於解決市面上記帳軟體功能不全的問題，並且提供一個具有完善功能的APP供使用者使用。為了實現這一目的，我們將學習和使用Flutter語法、Google Firebase和Git操作。此外，本軟體的特點包括：

    1. 提供共同帳本功能，使AA制的內容能夠更加得清楚明瞭。
    2. 將墊付和欠款的紀錄整理得清晰，方便追蹤。
    3. 將預算與實際金額的完成度對照，使資金的使用更加明確。

    開發架構:
    • 開發環境 : Android studio
    • 開發語言 : Flutter
    • 資料庫 : Google FireBase
    • 其他 : Git版本控制
